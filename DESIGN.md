# DevForge-CLI 项目设计方案

## 一、项目概述

**项目名称**: DevForge-CLI  
**项目类型**: CLI工具箱/零依赖Python工具集  
**核心定位**: 本地开发者瑞士军刀，提供30+常用开发工具，完全离线运行，保护隐私  
**灵感来源**: DevToys桌面应用（但专注于终端CLI场景）

## 二、技术选型

- **编程语言**: Python 3.8+
- **依赖管理**: 零外部依赖（仅使用Python标准库）
- **架构**: 单模块 + 子命令模式
- **界面**: CLI TUI（使用标准输出彩色格式化）

## 三、核心功能清单

### 数据转换类
1. **Base64编解码** - 文本/文件Base64加密解密
2. **URL编解码** - URL特殊字符处理
3. **HTML实体编解码** - HTML转义字符处理
4. **Unicode转义** - Unicode字符处理

### 格式处理类
5. **JSON格式化** - JSON美化/压缩
6. **YAML转换** - JSON↔YAML双向转换
7. **XML格式化** - XML美化/压缩
8. **TOML转换** - JSON↔TOML转换

### 加密安全类
9. **哈希计算** - MD5/SHA1/SHA256/SHA512
10. **UUID生成** - UUID v1/v4/v5生成
11. **密码生成** - 随机安全密码生成
12. **JWT解码** - JWT Token解析

### 网络时间类
13. **时间戳转换** - Unix时间戳↔日期时间
14. **Cron解析** - Cron表达式可视化
15. **时区转换** - 多时区时间转换
16. **URL解析** - URL结构分解

### 文本处理类
17. **正则测试** - 正则表达式实时测试
18. **文本差异** - 两段文本对比
19. **行计数** - 统计行/词/字符
20. **大小写转换** - 各种大小写格式

### 颜色格式类
21. **颜色转换** - HEX/RGB/HSL互转
22. **颜色预览** - 终端颜色预览

### 编码解码类
23. **Morse电码** - 莫尔斯电码编解码
24. **JWT编码** - JWT Token生成

### 实用工具类
25. **二维码生成** - 终端二维码显示
26. **密码强度检测** - 密码安全评分
27. **文件哈希** - 文件完整性校验
28. **随机数生成** - 各种随机数生成

## 四、技术架构

```
devforge/
├── main.py              # 主入口
├── utils/
│   ├── __init__.py
│   ├── base64_tool.py   # Base64工具
│   ├── hash_tool.py     # 哈希工具
│   ├── json_tool.py     # JSON工具
│   ├── yaml_tool.py     # YAML工具
│   ├── time_tool.py      # 时间工具
│   ├── regex_tool.py     # 正则工具
│   ├── color_tool.py     # 颜色工具
│   └── generator.py     # 生成器工具
├── README.md
├── requirements.txt     # 空文件（零依赖）
├── setup.py
└── .gitignore
```

## 五、差异化亮点

1. **零依赖** - 仅使用Python标准库，安装简单
2. **全离线** - 所有处理在本地完成，数据不外传
3. **跨平台** - Windows/macOS/Linux通用
4. **TUI界面** - 彩色终端输出，美观易读
5. **管道友好** - 支持管道输入输出
6. **中文优化** - 完整中文支持

## 六、自测标准

1. 所有工具命令可正常执行
2. 无导入错误
3. 输出格式正确
4. 帮助信息完整
5. 无安全漏洞
